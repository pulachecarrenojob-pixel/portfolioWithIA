# 🖤 Personal Portfolio — Frontend Developer

Modern portfolio built with vanilla HTML, CSS and JS. No frameworks, no dependencies.

## 🚀 Run it

Open `portfolio.html` directly in your browser, or use **Live Server** in VS Code.

## ✏️ Customize

Search for `✏️` comments in the HTML — they mark everything you need to change:

- **Name** — Hero, About card, Footer
- **Photo** — Replace the `👤` emoji with an `<img>` tag
- **Bio & Stats** — About section
- **Skills** — Add/remove `.pill` elements
- **Experience** — Edit each `.exp-item`
- **Projects** — Update titles, descriptions and links
- **Contact** — Your email, GitHub, LinkedIn

## 🎨 Change Colors

Edit the CSS variables at the top of the `<style>` block:

```css
:root {
  --bg: #0a0a0a;
  --text: #f0f0f0;
  --text2: #888;
}
```

## 🌐 Deploy

- **Vercel / Netlify** — drag & drop the file, done.
- **GitHub Pages** — rename to `index.html`, push to `yourusername.github.io`.
